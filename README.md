# vi-element-vue

# install 
``` 
# Using npm 
npm i vi-element-vue -S


# Using yarn 
yarn add vi-element-vue

```

# FileUrl
<https://luyibiao.github.io/vi/dist/#/>

# Quickstart
```
import Vue from 'vue';
import vi from 'vi-element-vue'
import 'vi-element-vue/lib/index.css'


Vue.use(vi)
```
